# ⚡ MACDERT TRAYS — DSA Problem Tracker

**MACDERT TRAYS** is a high-performance, visually stunning DSA tracker designed for the modern developer. It helps you track your journey through the **Striver A2Z Sheet**, **LeetCode**, **Codeforces**, and **CodeChef** with a gamified experience.

![Dashboard Preview](https://via.placeholder.com/800x450?text=MACDERT+TRAYS+Dashboard)

## 🚀 Key Features

- **Gamified Progress**: Earn XP, level up your Rank (Newbie to Grandmaster), and maintain a daily streak.
- **Comprehensive Sheet**: Track 750+ curated problems from various platforms.
- **Interactive Dashboard**:
    - **Progress Stats**: Real-time tracking of solved problems and ETA to completion.
    - **Topic Rings**: Visual representation of your progress in each DSA topic.
    - **Activity Heatmap**: 52-week view of your consistency.
- **Built-in Tools**:
    - **Pomodoro Timer**: Stay focused with customizable intervals.
    - **Hint Reminder**: Smart notification if you're stuck on a problem for over 30 minutes.
    - **CF Timer**: Dedicated timer for competitive programming practice.
- **Personalized Tracking**:
    - **Bookmarks**: Save difficult problems for later.
    - **Revision Mode**: Mark problems for future practice.
    - **Notes**: Store your approaches, complexity, and edge cases directly within the app.
- **Data Portability**: Easily **Export** and **Import** your progress as a JSON file.
- **Premium Aesthetic**: Sleek Cyberpunk/Neon UI with smooth animations and responsive design.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Custom properties, Flexbox/Grid, and Neon effects.
- **JavaScript (ES6+)**: Core logic, progress calculation, and state management.
- **Google Fonts**: Orbitron, Share Tech Mono, and Exo 2.

## 📂 Project Structure

- `index.html`: The main entry point and UI layout.
- `style.css`: The "soul" of the app, containing all the cyberpunk styling and animations.
- `app.js`: The "brain" of the app, handling all interactivity, timers, and progress logic.
- `data.js`: Contains the massive collection of 750+ problems and topic metadata.
